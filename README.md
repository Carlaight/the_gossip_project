### Clone notre repo et c'est parti !

- Postionne toi dans le dossier,
- `Bundle install`  # gem gem gem
- `rails db:create` # création de la BDD
- `rails server`
- Rejoins l'url `localhost:3000` grâce à ton navigateur préféré
